# ReLead Ops branding brief

## Name

ReLead Ops

## Tagline options

1. Secure infrastructure command center.
2. Operaciones seguras para infraestructura moderna.
3. Control plane for SSH, Codespaces and deployments.
4. Your infrastructure, safely orchestrated.

## Brand personality

- secure;
- technical;
- reliable;
- fast;
- clean;
- operational.

## Logo concept

A minimal shield containing a terminal prompt and connected nodes.

Suggested elements:

- shield outline for security;
- `>` prompt for command execution;
- three connected dots for targets/network;
- subtle lightning or pulse line for health monitoring.

## Logo generation prompt

Create a modern vector logo for “ReLead Ops”, a secure infrastructure operations platform. Use a minimal shield outline combined with a terminal prompt symbol and three connected network nodes. Style: clean SaaS, dark background, electric cyan and deep blue accents, high contrast, professional, scalable, no gradients that hurt legibility, no mockups, no 3D, centered composition.

## UI visual direction

- Dark dashboard.
- Sidebar navigation.
- Cards with subtle borders.
- Health indicators with status pills.
- Monospace command previews.
- Realtime logs in terminal-style panels.
